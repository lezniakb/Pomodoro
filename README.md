A simple Python timer to remind of a break

This repository has been created without the help of AI in any capacity.<br>
If you interact with this source code in any capacity, please respect the No-AI rule.

### Requirements
- [Python 3](https://www.python.org/downloads/)
- [winotify](https://pypi.org/project/winotify/) module
